Carbon Intensity API

This is the Official Carbon Intensity API for Great Britain developed by National Grid. You can find out more about Carbon Intensity at https://carbonintensity.org.uk.

National Gridâ€™s Carbon Intensity API provides an indicative trend of carbon intensity of the electricity system in Great Britain (GB) up to 48-hours ahead of real-time. It provides programmatic and timely access to both estimated and forecast carbon intensity data. It includes the CO2 emissions from all large metered power stations and interconnector imports, whilst accounting for national demand, embedded wind and solar generation. This report details the methodology behind the estimated and forecast carbon intensity data streams.

It is important to note that these forecasts are likely to be less accurate than other forecasts produced by National Grid, since they include the confluence of uncertainties in demand, wind, solar, and the uncertainties in the forecasts for each fuel type (gas, coal, biomass, interconnectors etc.). This approach does not use Physical Notifications (PNs) of Balancing Mechanism (BM) units in its forecasts. This is to ensure that the commercial sensitivities surrounding the balancing market are maintained.

get /intensity/date/{date}
Get Carbon Intensity data for specific date (intensityDateDateGet)
Get Carbon Intensity data for a specific date. All times provided in UTC (+00:00).

Path parameters

date (required)
Path Parameter — Date in YYYY-MM-DD format e.g. 2017-08-25

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Return type

Intensity

Content-Type: application/json

  },
  "from" : "aeiou",
  "to" : "aeiou"
}
