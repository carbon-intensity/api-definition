    GET /intensity/stats/{from}/{to}

    Get emission statistics for a requested period

    Parameters

    Parameter In Type Required Description
    from path string(datetime) true Start date and time in ISO8601 format, e.g. 2017-09-01T00:00Z
    to path string(datetime) true End date and time in ISO8601 format, e.g. 2017-09-02T12:00Z

    Responses

    Status Meaning Description Schema
    200 OK No description None
    400 Bad Request No description None
    This operation does not require authentication

    Schemas

    IntensityArray

    {
  "data": [
    {
      "from": "2017-09-12 00:30",
      "to": "2017-09-12 01:00",
      "intensity": {
        "actual": 0,
        "forecast": 137.1,
        "index": "moderate"
      }
    }
  ]
}

    Properties

    Name Type Required Description
    data [IntensityValue] false No description
    » from string(datetime) false The beginning of the half hour settlement period. In the format YYYY-MM-DD HH:MM and is the local time so either GMT(+0) or BST(+1) depending on Daylight Savings Time.
    » to string(datetime) false The end of the half hour settlement period. In the format YYYY-MM-DD HH:MM and is the local time so either GMT(+0) or BST(+1) depending on Daylight Savings Time.
    » intensity object false No description
    »» actual number(float) false The average outturn CO2 emission intensity or carbon intensity for the half hour settlement period with units gCO2-eq/kWh. For future values, this will be null.
    »» forecast number(float) false The average forecast CO2 emission intensity or carbon intensity for the half hour settlement period with units gCO2-eq/kWh.
    »» index string false How high the emission value is, compared to average. Can be one of "very high", "high", "moderate", "low", or "very low".
